Gmail Setup Guide

To use Gmail for sending notifications, you need to create an App Password:

  1. Go to Google Account Security
  2. Enable 2-Factor Authentication if not already enabled
  3. Go to App Passwords
  4. Create a new app password for "Mail"
  5. Use the 16-character password generated (without spaces) below

Important: Never use your regular Gmail password. Always use an App Password!
